AI-powered real-time public safety surveillance command-center prototype with live detection, person tracking, SOS emergency alerts, authority response workflow, heatmap analytics, and AI assistant support.

Team: TriNetra

Platform: Local Windows Prototype
Repository: GitHub Uploaded

01 Final Review Note

We, Team TriNetra, have completed the final MVP of our project SurakshaDrishti AI.

SurakshaDrishti AI is a local real-time AI surveillance command-center prototype designed to support public safety monitoring. It uses live webcam input, AI object detection, person tracking, SOS emergency alert generation, pinned emergency alerts, authority response workflow, heatmap analytics, and a local AI assistant.

The system has been successfully tested. Backend, frontend dashboard, live feed, detection, tracking, SOS, authority workflow, heatmap, and assistant are working properly.

GitHub Repository:
https://github.com/maheshofficial0011/SurakshaDrishti-AI

Status: Final MVP completed, tested, documented, uploaded to GitHub, and ready for final demo/submission.

05 Authority Response Workflow

PENDING
→
ASSIGNED
→
RUNNING / DISPATCHED
→
RESOLVED
  • PENDING: Incident created, no unit assigned yet.
  • ASSIGNED: Unit selected for response.
  • RUNNING: Unit is responding.
  • RESOLVED: Incident completed.
